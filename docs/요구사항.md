개발 직군 과제 테스트
📌 과제 개요
간단한 영화 티켓 예매 시스템을 구현하세요.
🧰 필수 기술 스택
Server: Node.js
Database: PostgreSQL
※ 그 외 기술 스택은 자유롭게 선택 가능합니다. (Framework, ORM, Frontend 등)
🎯 요구 기능
1. 회원 기능
   회원가입
   로그인
2. 영화 / 상영 정보
   영화 목록 조회
   상영 시간 목록 조회
   ※ 영화 및 상영 데이터는 실제 데이터일 필요 없으며, 임의 데이터로 구성해도 무방합니다.
3. 좌석 예매
   좌석 선택 및 예매 기능
   예매 내역 조회
